Welcome to CustomTools!

  • CustomTools is an extension of pyRevit add-in for Autodesk Revit.
  • This is MassMessage window which is meant to serve as an internal news channel for all Revit users inside your company or group of users. Change the URL to your custom HTML file using customToolsSettings.
  • You can use it for sharing new families, templates, scripts, learning content etc. with your team.

After the installation

  • Use pyRevit Search (CTRL+F), write customToolsSettings and press Enter
  • Set paths for all features which are you going to use
  • Set Power BI reports to view logs. Templates are available via Shift + Click on the Dashboards icon.

Help

  • Brief description of the tools is available as tooltips. You need to hold your cursor on selected button to show the tooltip. If there is F1 sign in the tooltip you can press F1 button on your keyboard to access the video manual.
